# Contracts & Disputes Module

> **DISCLAIMER:** All contract documents generated by this module are
> templates for internal business use only and do not constitute legal advice.
> Have a licensed attorney review all agreements before execution.

---

## Overview

The `contracts/` package provides three core services for the
Pay-Per-Qualified-Lead platform:

| Module | Class | Purpose |
|---|---|---|
| `contract_templates.py` | `ContractGenerator` | Generate MSA, addenda, and delivery receipts |
| `dispute_engine.py` | `DisputeEngine` | Evaluate and log lead disputes deterministically |
| `billing_protection.py` | `BillingProtection` | Pre-charge validation, invoice generation, volume discounts |
| `contracts_streamlit_tab.py` | `render_contracts_tab` | Streamlit UI tab |

## 1. Generating and Sending Contracts

### Quick Start

```python
from contracts import ContractGenerator

gen = ContractGenerator()

buyer_info = {
    "company_name": "Acme Lending LLC",
    "contact_name": "Jane Smith",
    "address": "123 Main St, Tampa FL 33601",
    "email": "jane@acmelending.com",
    "state": "Florida",
}

# Full Master Service Agreement
msa = gen.generate_msa(buyer_info, vertical="real_estate")

# Vertical-specific addendum
addendum = gen.generate_vertical_addendum(
    vertical="real_estate",
    sub_vertical="bridge_loan",
    buyer_info=buyer_info,
)

# Lead delivery receipt
receipt = gen.generate_lead_delivery_confirmation(
    lead_data={
        "lead_id": "LEAD-001",
        "name": "John Doe",
        "phone": "555-123-4567",
        "property_address": "456 Ocean Dr, Miami FL 33139",
        "lead_type": "Bridge Loan",
        "zip_code": "33139",
        "delivered_at": "2024-01-15 14:30:00 UTC",
        "qualification_criteria_met": [
            "Deal identified",
            "Funds verified",
            "Timeline within 120 days",
        ],
    },
    buyer_info=buyer_info,
    charge_amount=250.00,
)
```

### Sending Contracts

1. Generate the contract text using `ContractGenerator`.
2. Deliver via the **Contracts & Disputes** tab in the Streamlit app (download
   as Markdown or TXT).
3. Send to the buyer via your e-signature tool (e.g., DocuSign, HelloSign)
   for execution.
4. Store the signed copy in your CRM.

## 2. Dispute Resolution SLA and Process

### SLA

| Step | Timeline |
|---|---|
| Buyer submits dispute | Within **72 hours** of delivery |
| Auto-decision (eligible disputes) | **Immediate** |
| Manual review decision | Within **5 business days** |
| Credit applied to invoice | On next monthly invoice (1st of month) |
| Replacement lead issued | Within **10 business days** of approval |

### Dispute Reasons and Handling

| Reason | Auto-approved? | Condition |
|---|---|---|
| `wrong_number` | ✅ Yes | Call duration < 30 s on record |
| `duplicate` | ✅ Yes | Lead delivered twice to same buyer |
| `outside_territory` | ✅ Yes | Zip not in buyer's territory config |
| `not_qualified` | ❌ No — manual review | Human QA verification |
| `fake_lead` | ❌ No — manual review | Human QA verification |

### Process

```python
from contracts import DisputeEngine
from contracts.dispute_engine import DisputeSubmission
from datetime import datetime, timezone

engine = DisputeEngine()

# Configure engine data (done at startup / from database in production)
engine.register_delivery("LEAD-001", "BUYER-42", price=250.00)
engine.register_call_duration("LEAD-001", duration_seconds=15)

submission = DisputeSubmission(
    lead_id="LEAD-001",
    buyer_id="BUYER-42",
    buyer_email="buyer@example.com",
    dispute_reason="wrong_number",
    submitted_at=datetime.now(timezone.utc),
    lead_delivered_at=datetime(2024, 1, 15, 14, 30, tzinfo=timezone.utc),
)

decision = engine.evaluate_dispute(submission)
print(decision.decision)       # "auto_approved"
print(decision.credit_amount)  # 250.0
```

## 3. Invoice Workflow

### Monthly Invoice Generation

Invoices are generated on the **1st of each month** covering the prior
calendar month.

```python
from contracts import BillingProtection
from datetime import datetime, timezone

billing = BillingProtection()

billing.register_buyer(
    buyer_id="BUYER-42",
    buyer_info={"company_name": "Acme Lending LLC"},
    payment_method="stripe_pm_xxxxx",
    monthly_cap=200,
)

# Pre-charge validation
ok, reason = billing.validate_before_charge(
    lead_data={"phone": "555-123-4567", "email": "lead@example.com"},
    buyer_id="BUYER-42",
)
if not ok:
    print(f"Charge blocked: {reason}")

# Generate invoice
invoice = billing.generate_invoice(
    buyer_id="BUYER-42",
    period_start=datetime(2024, 1, 1, tzinfo=timezone.utc),
    period_end=datetime(2024, 1, 31, tzinfo=timezone.utc),
    deliveries=[
        {
            "lead_id": "LEAD-001",
            "lead_type": "Bridge Loan",
            "delivered_at": datetime(2024, 1, 10, tzinfo=timezone.utc),
            "amount": 250.00,
        }
    ],
)
print(invoice["total_due"])  # 250.0 (or less if credits applied)
```

### Volume Discounts

Discounts are calculated by `BillingProtection.calculate_volume_discount()`:

| Monthly Leads | Discount |
|---|---|
| 0 – 99 | 0% |
| 100 – 249 | 0% |
| 250 – 499 | 11% |
| 500 – 999 | 22% |
| 1,000+ | 32% |

## 4. Stripe Integration

To automate billing, integrate `BillingProtection` with Stripe:

### Setup

```bash
pip install stripe
```

### Charge on Lead Delivery

```python
import stripe
stripe.api_key = "sk_live_..."

def charge_buyer(buyer_id: str, payment_method_id: str, amount_usd: float):
    """Charge buyer immediately on lead delivery."""
    stripe.PaymentIntent.create(
        amount=int(amount_usd * 100),  # Stripe uses cents
        currency="usd",
        payment_method=payment_method_id,
        confirm=True,
        description=f"Lead delivery — buyer {buyer_id}",
    )
```

### Monthly Invoice via Stripe Billing

```python
def create_stripe_invoice(buyer_stripe_customer_id: str, invoice: dict):
    """Push a generated invoice dict to Stripe as an invoice."""
    for item in invoice["line_items"]:
        stripe.InvoiceItem.create(
            customer=buyer_stripe_customer_id,
            amount=int(item["amount"] * 100),
            currency="usd",
            description=f"{item['lead_type']} lead — {item['lead_id']}",
        )
    if invoice["credits_applied"] > 0:
        stripe.CustomerBalanceTransaction.create(
            customer=buyer_stripe_customer_id,
            amount=-int(invoice["credits_applied"] * 100),
            currency="usd",
            description="Lead dispute credit",
        )
    stripe_invoice = stripe.Invoice.create(
        customer=buyer_stripe_customer_id,
        auto_advance=True,
    )
    stripe.Invoice.finalize_invoice(stripe_invoice.id)
```
